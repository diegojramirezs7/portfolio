Sorting Algorihm Visualizer

Traditional Sorting Algorithm Visualizer.

Project Description

React-based web application that visualizes 6 of the most common sorting algorithms. The algorithms implemented are: Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Heap Sort, and Quick Sort. The website allows you to visualize 1, 2, or 4 algorithms at a time, which is handy for comparing their running time.

This project was almost fully built with React.JS. All elements on the screen are React components that follow the fundamental React logic. The algorithms and animations were implemented by updating the state of the arrays. For the theming and styling of the website I used the Material-UI framework, which is a popular React UI Framework.

Development Process

I built this project on my own after a some weeks of learning React. Given that React is becoming one of the most popular Javascript Frameworks I decided to start learning it by building some cool projects along the way.
